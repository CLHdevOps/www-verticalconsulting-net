## Tips for Best Results

1. **Diversity**: Vary gender, ethnicity, and age across the three images to reflect diverse clientele
2. **Consistency**: Use similar lighting and background styles for visual cohesion
3. **Professionalism**: Ensure subjects are dressed appropriately for enterprise IT consulting context
4. **Expression**: Mix of confident smiles and professional serious expressions
5. **Quality**: Generate at minimum 1024x1024px then crop/resize to 400x400px

## How to Generate

### Using DALL-E (ChatGPT Plus or API):
1. Go to ChatGPT and paste one of the prompts above
2. Download the generated image
3. Resize to 400x400px if needed
4. Save with the specified filename

### Using Midjourney:
1. Use the `/imagine` command with the prompt
2. Add `--ar 1:1` for square aspect ratio
3. Upscale your favorite result
4. Download and rename appropriately

### Using Stable Diffusion:
1. Use any SD interface (Automatic1111, ComfyUI, etc.)
2. Paste the prompt
3. Set dimensions to 512x512 or 1024x1024
4. Generate and save
